## How to Use

### Starting the Demo

1. **Navigate to Demo Page**
   - Click "Complete Demo" in sidebar
   - Or go to `/demo`

2. **Review Current Step**
   - Read the step title and description
   - Review actions to perform
   - Check expected results

3. **Navigate to Target Page**
   - Click "Navigate" button
   - This opens the actual page in a new tab/window
   - Perform the actions listed

4. **Mark Complete**
   - Click "Mark Complete" when done
   - Or click "Next Step" to continue

### Auto-Play Mode

1. **Enable Auto-Play**
   - Click "Auto-Play" button
   - Steps will advance automatically
   - Duration shown for each step

2. **Pause/Resume**
   - Click "Pause" to stop auto-advance
   - Click "Play" to resume

3. **Manual Control**
   - You can still navigate manually
   - Auto-play respects manual navigation

### Navigation Options

1. **Previous/Next Buttons**
   - Use "Previous" to go back
   - Use "Next Step" to advance
   - Buttons disabled at boundaries

2. **Jump to Step**
   - Use sidebar "All Steps" list
   - Click any step to jump directly
   - Completed steps shown in green

3. **Section Navigation**
   - View section progress in sidebar
   - See which sections are complete
   - Current section highlighted

## Troubleshooting

### Demo Page Not Loading
- **Check authentication**: Must be logged in
- **Check URL**: Should be `/demo`
- **Refresh page**: Try hard refresh (Ctrl+F5)

### Navigation Not Working
- **Check target page exists**: Some pages may need setup
- **Check permissions**: Some features require specific roles
- **Try direct URL**: Copy page URL and navigate manually

### Steps Not Advancing
- **Check auto-play**: Make sure it's enabled if using
- **Manual navigation**: Use Previous/Next buttons
- **Jump to step**: Use sidebar to jump directly

### Progress Not Saving
- **Check browser**: Uses localStorage
- **Check privacy settings**: Some browsers block localStorage
- **Progress resets**: On page refresh (by design for fresh demos)
